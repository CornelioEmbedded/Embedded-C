## Escape sequences

- `\'` Single Quote
- `\"` Double Quote
- `\\` Backslash
- `\0` Null
- `\a` Bell
- `\b` Backspace
- `\f` Form feed
- `\n` Newline
- `\r` Carriage return
- `\t` Horizontal Tab
- `\v` Vertical tab
